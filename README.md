# Rhythmic

**Rhythmic** is a playlist manager that allows you to manage playlists across multiple music streaming providers seamlessly.

## Features

- Create, edit, and organize playlists across different platforms.
- Sync playlists between popular music providers.
- Supports multiple accounts per provider.
- Transfer playlists between platforms.

## Supported Platforms

- Spotify
- Apple Music
- YouTube Music
- Amazon Music
- Soundcloud
- Deezer
